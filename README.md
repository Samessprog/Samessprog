## 🚀 About Me

I build environments that let developers push code and sleep well at night. I transitioned from a strong **Fullstack** background to **DevOps & Cloud Engineering** because I care deeply about how systems scale, survive traffic spikes, and deploy seamlessly. I specialize in Infrastructure as Code (IaC), event-driven architectures, and CI/CD automation.

* 🔭 **Currently architecting:** [Kaizen Box Nexty](https://github.com/Samessprog/Nexty) – an AI-powered SaaS with an event-driven backend. Provisioning the entire AWS infrastructure (ECS, RDS, Bedrock, SQS/SNS) 100% via **Terraform** and **Ansible**.
* 💼 **Day Job:** DevOps Engineer at MindPal. I manage AWS/GCP infrastructure, migrate legacy systems (recently slashed cloud spend by **80%** for 8 clients), and optimize CI/CD pipelines with Playwright.
* 🏠 **Home Lab:** I maintain a custom self-hosted cloud on a Raspberry Pi 5. Bypassed ISP double NAT using a **Tailscale mesh VPN**, running Dockerized Nextcloud, Pi-hole, and automated Bash backups.
* 🤖 **AI & Data:** Completing my Master's Thesis in **Deep Learning & Computer Vision** (PyTorch/ResNet) and actively integrating Generative AI (AWS Bedrock) into serverless data pipelines (AWS Glue, Athena).
